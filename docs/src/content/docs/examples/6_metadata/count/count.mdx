---
title: Count
description: Setting default record counts with metadata
---


### Model

```go title="orders.dg"
model orders {
  metadata {
    count: 5
  }

  fields {
    id() string
    amount() float64
  }

  gens {
    func id() {
      return UUID()
    }

    func amount() {
      return FloatBetween(10.0, 250.0)
    }
  }
}
```


### Usage Examples

#### 1. Use model default (metadata.count)

```shell
$ datagenc gen ./examples/6_metadata/count
```

**Output:**

#### 2. Override default for all models

```shell
$ datagenc gen ./examples/6_metadata/count -n 3
```

**Output:**
